WSDL2Kotlin
--------------------

A stub generator for services implemented by WSDL.

Inspired by [WSDL2Swift](https://github.com/banjun/WSDL2Swift)

# for developers

## build


```
$ ./gradlew build
```

## test


```
$ ./gradlew test
```

## lint

check your source code:

```
$ ./gradlew ktlint
```

format all source code automatically:

```
$ ./gradlew ktFormat
```
